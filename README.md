## 🚀 I Miei Progetti Principali

🌱 **[EcoFlora (CitizenScience)](https://github.com/luke03b/CitizenScience)**
*Un'applicazione multi-container per la registrazione e l'identificazione di fiori selvatici sul campo*.
* **Stack:** Flutter (Frontend web), Spring Boot (Backend REST API), FastAPI con PyTorch (Servizio AI basato su ResNet18), PostgreSQL con PostGIS (Database spaziale).
* **Architettura & DevOps:** Gestito tramite Docker Compose con Nginx come reverse proxy. Include pipeline CI/CD tramite GitHub Actions per test, CodeQL e SonarQube per l'analisi del codice.

🏠 **[House Hunters (App)](https://github.com/luke03b/INGSW2425_026) & [Backend](https://github.com/luke03b/HouseHuntersBackEnd)**
*Sistema client-server per la gestione di annunci immobiliari.*
* Applicazione client multipiattaforma sviluppata interamente in Flutter/Dart.
* Architettura backend dedicata scritta in Java.

📦 **[UninaDelivery_09](https://github.com/luke03b/UninaDelivery_09)**
*Software gestionale aziendale.*
* Applicazione sviluppata in Java per un'azienda di vendita e spedizione di merci.

🏛️ **[MemeMuseum](https://github.com/luke03b/MemeMuseum)**
*Piattaforma web containerizzata.*
* Progetto full-stack (Frontend e Backend) gestito tramite `docker-compose`, interamente sviluppato in TypeScript, HTML e SCSS.

📼 **[LsoVideoteca](https://github.com/luke03b/LsoVideoteca)**
*Progetto di livello sistema orientato alla concorrenza.*
* Server scritto in C e PLpgSQL per la gestione di una videoteca.
* Implementa meccanismi avanzati di sincronizzazione e gestione della concorrenza tramite *mutex*.
